1. 샤프 최적화에서 중기를 Look Back 할 떄가 성과가 제일 좋았다(샤프가 단봉으로 그려짐) -> 주식은 단기 레짐 영향을 많이 받는다 + 모멘텀 / max_sharpe=0.63 
	S&P를 아웃퍼폼한게 나왔지만, 통계적으로 유의하지는 않았다 / 포지션 크게 잡음 -> 근데 오히려 이게 weight 집중이 다른 것보다 덜 한 것 같노

2. 1번과 똑같은 조건에서 max_ratio 10%와 20%를 실험했지만 오히려 성과는 이전보다 감소했다 ***이거 Look-Back Window 1번이랑 맞춰서 다시 돌리기***

3. constant는 쓰래기 / linear는 비슷 / eigenvalue clipping은 성과가 좋았음 -> 클립핑에서 샤프 0.7찍었음(통계적으로도 10%에서 마켓 아웃퍼폼함)
 -> None이랑 비교했을떄, Clipping은 포지션을 비슷하게 잡지만, 조금 덜 잡아서 분산이 좀 더 되는 효과가 있다

4. 목적함수가 분산으로 들어가면, 확실히 성과가 룩벡윈도우에 크게 영향을 안받음 (추정하는데 리턴이 안 포함되어서 그런 듯)
 -> 분산 잘 되는데, 한번씩 너무 크게 잡는다.

5. 최소 분산도 Eigen Value Clipping이 성과가 가장 좋다 / Look-Back Window가 짧을 수록 분산은 오히려 더 잘 안된다

6. clipping 값은 5일때 성과가 가장 좋았다(이건 샤프에서도 마찬가지)
